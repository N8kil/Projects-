# 🦟 Dengue Prediction using Machine Learning

This project predicts dengue cases using weather and environmental features with **machine learning models**.  
It demonstrates **data preprocessing, feature engineering, model training, and hyperparameter tuning**.

---

## 📂 Project Structure
- `data/` → Dengue features and labels (train/test CSV files)  
- `notebooks/` → Jupyter notebook with full analysis and model training  
- `README.md` → Documentation  

---

## 🚀 Workflow
1. Data preprocessing (handling missing values, scaling, encoding)  
2. Feature engineering (weather patterns, time-based features)  
3. Model training (Random Forest, Gradient Boosting, etc.)  
4. Hyperparameter tuning (GridSearchCV)  
5. Model evaluation (MAE, RMSE, R² Score)  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn  

---

## 📊 Results
- Random Forest achieved **Mean Absolute Error (MAE): ~16**  
- GridSearchCV improved results with optimized hyperparameters
